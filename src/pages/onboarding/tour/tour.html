Key features of MiraBox

Simplicity

It's easy to transfer and store, because this is a regular file
Clear smart contracts

Conditions for opening and working with Mirabox
Security

Advanced double-encryption of the MiraBox file

There are three types of MiraBox containers

Nominal Box - a container with one type of cryptocurrency included. MultiBox - a container with configurable number of different supported cryptocurrencies. SmartBox - MultiBox with integrated smart contracts that allow the setting of various conditions for opening and extracting (currency rate, third-party confirmation, specific date, etc).
